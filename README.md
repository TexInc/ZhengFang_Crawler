# ZhengFang_Crawl
最新Python3.6正方教务系统模拟登陆，获取信息,目前已测试通过

### 项目依赖

- Python3.6
- bs4
  - 用于对网页做解析
- request
  - 模拟http请求
- PIL
  - 操作图片，主要是验证码
- urllib
  - 主要用来对url进行更改操作





### 主要文件  



#### main.py
> 主程序，用于对爬虫功能进行调用    




#### get_operate.py
> 获取所有可以进行的操作链接  




#### crawl_func.py
> 主要爬虫的功能
